
# 📊 SQL Data Warehouse Project

## 📌 Project Overview

Welcome to the SQL Data Warehouse Project!

This project demonstrates the design and implementation of an end-to-end Data Warehouse using Microsoft SQL Server. It follows the Medallion Architecture (Bronze, Silver, and Gold) to extract, clean, transform, and model data into analytics-ready datasets.

The project is designed to simulate a real-world data warehouse environment and follows industry best practices for ETL development and dimensional modeling.

---

## 🎯 Project Objectives

- Build a scalable SQL Data Warehouse.
- Implement the Medallion Architecture.
- Develop ETL pipelines.
- Clean and validate raw data.
- Design a Star Schema.
- Create Fact and Dimension tables.
- Prepare data for Business Intelligence and Reporting.

---

## 🏗️ Architecture

The project follows the Medallion Architecture:

### 🥉 Bronze Layer
- Load raw data from source systems.
- Preserve original data.
- No transformations.

### 🥈 Silver Layer
- Clean data.
- Handle missing values.
- Remove duplicates.
- Standardize formats.
- Apply business rules.

### 🥇 Gold Layer
- Create analytical models.
- Build Fact and Dimension tables.
- Prepare data for reporting and dashboards.
